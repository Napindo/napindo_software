# Admin User Flow

Dokumen ini menjelaskan alur kerja pengguna Admin pada aplikasi Napindo Software.

## Peran dan Akses
- Akses penuh ke semua halaman.
- Boleh import data, delete data, melihat audit log, dan menambah user.
- Bisa membuka fitur Search di halaman Add Data.

## Alur Utama
1. Login
   - Masuk dengan username dan password Admin.
2. Dashboard
   - Melihat ringkasan awal dan daftar user online.
3. Input Data
   - Add Data: isi form, simpan data baru (Add/Save).
   - Import Data: upload file sesuai format yang didukung untuk batch insert.
4. Search Data
   - Exhibitor / Visitor: cari, pilih baris, lalu Edit untuk membuka Add Data dengan field terisi.
   - Update/Edit: lakukan perubahan dan simpan.
5. Print Label
   - Perusahaan / Government: pilih filter, preview, lalu export.
6. Report
   - Perusahaan / Government / Jumlah: pilih filter dan export.
7. User Management
   - Log: lihat aktivitas user dan status online.
   - Add User: buat akun baru dan set divisi.
   - Change Password: ubah password user.
8. Logout

## Catatan Operasional
- Edit data dari Search akan membuka Add Data dengan data terisi dan fokus ke Update/Edit.
- F5 untuk Add/Save dan F2 untuk Update/Edit bisa digunakan sesuai konteks.
