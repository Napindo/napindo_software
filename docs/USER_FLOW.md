# User Biasa Flow

Dokumen ini menjelaskan alur kerja pengguna biasa (akses terbatas) pada aplikasi Napindo Software.

## Peran dan Akses
- Halaman yang tersedia: Add Data, Exhibitor, Visitor, Change Password.
- Tidak bisa import data, delete data, melihat audit log, atau menambah user.
- Fitur Search di halaman Add Data tidak tersedia.

## Alur Utama
1. Login
   - Masuk dengan username dan password yang diberikan admin.
2. Add Data
   - Isi form dan simpan data baru (Add/Save).
   - Jika perlu update, buka data lewat Search (Exhibitor/Visitor) lalu Edit.
3. Search Data
   - Exhibitor / Visitor: cari data, pilih baris, lalu Edit untuk mengarahkan ke Add Data.
   - Update/Edit digunakan untuk menyimpan perubahan.
4. Change Password
   - Ganti password pribadi secara berkala.
5. Logout

## Catatan Operasional
- Tombol dan shortcut update hanya aktif ketika data sudah dipilih dari Search.
- Jika tidak menemukan data pada Search, buat data baru dari Add Data.
